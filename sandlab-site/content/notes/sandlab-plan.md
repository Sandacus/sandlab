Start at the start.
What do I want to acheive?
List of requirements for sandlab;

* Media Server
* Network attached storge (NAS)
* Selfhosted photos
* External access, VPN
* Offline internet (Wikipedia, Maps & other cool or useful stuff)
* Split up local network into subnets/vlans.
* Backup service for my computers, media and photos
* Home automation
* Infrastructure as code
* Selfhosted git server
