1. Download (Open Media Vault)[https://www.openmediavault.org/download.html]

2. Create a bootable USB drive with (Rufus)[https://rufus.ie/en/] or (Balena Etcher)[https://etcher.balena.io/]

3. Boot from USB and follow the OMV instructions to install

4. Use (OMV docs)[https://docs.openmediavault.org/en/8.x/administration/index.html] to mount storage devices and file systems
